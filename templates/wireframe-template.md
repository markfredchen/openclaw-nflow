## 交互说明

| 数据卡片 | 点击 | 跳转至详情页 |
| 刷新 | 下拉 | 调用 API 刷新数据 |

## 响应式设计

| 断点 | 宽度 | 布局变化 |
|------|------|----------|
| Mobile | < 768px | 单列，全宽卡片 |
| Tablet | 768-1024px | 双列网格 |
| Desktop | > 1024px | 三列，侧边栏固定 |

**Desktop 布局:**
```

│  [顶部导航栏]                                              │
```
