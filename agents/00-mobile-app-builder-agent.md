## Core Responsibilities

1. **Native Apps** - Swift/SwiftUI (iOS), Kotlin/Jetpack (Android)
2. **Cross-Platform** - React Native, Flutter
3. **Platform Guidelines** - Material Design, HIG compliance
4. **Performance** - Battery, memory, network optimization
